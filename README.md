# Onespace

> Україномовна платформа для пошуку фільмів та мультфільмів — з поділом на дитячий та дорослий контент, рейтингами IMDb і темним кінематографічним інтерфейсом.

Не стрімінг. Знайшов — перейшов на Netflix/Kino UA.

### Що в sitemap.md

- **Сутності** — 4 підтверджених об'єкти (Контент, Пошук+Фільтри, Дитячий режим, Добірка) + 3 під питанням
- **Навігація** — 3 глобальних пункти, таблиця глибини тапів, рівні (глобальна/контекстна/глибока)
- **Нові екрани з flows** — Налаштування PIN, Реєстрація/Вхід
- **Трасування** — матриця 11 jobs × 8 екранів, екрани-сироти і jobs-сироти з рішеннями

### Що в flows.md

- **Main Job** — повний flow від запуску до OTT: browse-шлях і search-шлях, empty/error стани, тупики
- **J2** — дитячий режим: setup PIN (перший раз), введення PIN (повторно), вихід з PIN
- **J3** — пошук з фільтрами: текстовий запит і панель фільтрів, empty стани, рафайн

## Ключові рішення

- **Дані**: TMDB API + IMDb datasets
- **Режими**: Дорослий / Дитячий з PIN-перемикачем
- **Freemium**: пошук безкоштовно → watchlist + дитячий режим платно
- **Стек**: Next.js 14 · Tailwind · PostgreSQL · Prisma · NextAuth · Vercel
- **Платформа**: Desktop → Mobile
