# Onespace

Україномовна платформа для пошуку фільмів та мультфільмів з поділом на дитячий та дорослий контент.

**Не стрімінг** — знаходьте фільми, дивіться рейтинги IMDb, переходьте на Netflix/Kino UA.

## Ключові можливості

- Швидкий пошук з фільтрами (жанр, рік, рейтинг, OTT-платформа)
- Два режими: Дорослий / Дитячий з PIN-захистом
- Тренди та топові підбірки
- Watchlist для зареєстрованих користувачів (Premium)

## Стек

Next.js 14 · Tailwind CSS · PostgreSQL · Prisma · NextAuth.js · TMDB API · IMDb datasets

## Статус

MVP у розробці (Desktop → Mobile)
