# User Flows — Onespace

> Вузли-екрани `[у квадратних дужках]` — відповідають sitemap.md.
> Рішення `{у фігурних дужках}` — розгалуження.
> Стани `[Стан: ...]` — окремі вузли всередині існуючого екрана (не нові екрани).
> `([...])` — термінальні точки: успіх або тупик.

---

## Main Job — Знайти фільм і перейти до перегляду

> Персони: всі три (Оксана, Дмитро, Ірина).
> Два шляхи: browse (без запиту) і search (з запитом). Обидва сходяться на Картці фільму.

```mermaid
flowchart TD
    START([Відкрив застосунок]) --> HOME[Головна]

    HOME --> LOAD{Контент\nзавантажився?}
    LOAD -->|так| FEED[Добірки і тренди на Головній]
    LOAD -->|збій| ERR[Стан: помилка мережі]
    ERR --> RETRY[Оновити]
    RETRY --> HOME

    FEED --> INTENT{Є конкретна\nназва або жанр?}
    INTENT -->|ні — гортаю| BROWSE[Гортаю добірки]
    INTENT -->|так — шукаю| SEARCH[Пошук]

    BROWSE --> MORE{Хочу більше\nодного жанру?}
    MORE -->|ні| TAP_B[Тап на постер у добірці]
    MORE -->|так| FULL[Повний список добірки]
    FULL --> TAP_B

    SEARCH --> QUERY[Вводжу запит або застосовую фільтри]
    QUERY --> HAS_R{Є результати?}
    HAS_R -->|так| TAP_S[Тап на постер у результатах]
    HAS_R -->|ні| EMPTY[Стан: empty — нічого не знайшлось]
    EMPTY --> FIX[Змінюю запит або фільтри]
    FIX --> QUERY
    EMPTY --> GIVE_UP([⚠️ Тупик: пішов з Пошуку без результату])

    TAP_B --> CARD[Картка фільму]
    TAP_S --> CARD

    CARD --> TRUST{Рейтинг і опис\nпереконали?}
    TRUST -->|ні — не моє| BACK[Назад до списку]
    BACK --> BROWSE
    TRUST -->|так| HAS_OTT{Є посилання\nна платформу?}
    HAS_OTT -->|так| OTT[Перехід на OTT-платформу]
    HAS_OTT -->|ні| NO_OTT[Стан: OTT недоступно]
    NO_OTT --> COPY[Копіює назву — шукає самостійно]

    OTT --> SUCCESS([✅ Успіх: дивиться на OTT])
    COPY --> DEADEND([⚠️ Тупик: вийшов шукати в Google])
```

## J2 — Безпечний простір для дитини

> Персона: Оксана (primary).
> Три сценарії: перший запуск (налаштування PIN), повторний візит (введення PIN), вихід з режиму (підтвердження PIN).

```mermaid
flowchart TD
    START([Оксана хоче дати дитині телефон]) --> HOME[Головна — дорослий режим]
    HOME --> TAP[Тап «Дітям» у навігації]

    TAP --> ISSET{PIN\nналаштовано?}
    ISSET -->|ні — перший раз| SETUP[Налаштування PIN]
    ISSET -->|так| ENTER[PIN-екран: введення]

    SETUP --> HAS_ACC{Є акаунт?}
    HAS_ACC -->|ні| REG[Реєстрація / Вхід]
    REG --> REG_OK{Реєстрація\nуспішна?}
    REG_OK -->|так| SETUP
    REG_OK -->|скасувала| NO_ACC([⚠️ Тупик: PIN не налаштовано — режим не активний])
    HAS_ACC -->|так| PIN_IN[Вводжу PIN і підтверджую повторно]
    PIN_IN --> MATCH{Обидва\nвведення збіглись?}
    MATCH -->|так| KIDS_HOME[Головна — дитячий режим]
    MATCH -->|ні| ERR_MATCH[Стан: PIN не збігається]
    ERR_MATCH --> PIN_IN

    ENTER --> CHECK{PIN\nправильний?}
    CHECK -->|так| KIDS_HOME
    CHECK -->|ні| ERR_PIN[Стан: неправильний PIN]
    ERR_PIN --> ATTEMPTS{3 спроби\nвичерпано?}
    ATTEMPTS -->|ні| ENTER
    ATTEMPTS -->|так| BLOCKED[Стан: заблоковано]
    BLOCKED --> SUPPORT([⚠️ Тупик: звернутись до підтримки])

    KIDS_HOME --> KIDS_BROWSE[Гортаю дитячі добірки]
    KIDS_BROWSE --> KIDS_CARD[Картка фільму]
    KIDS_CARD --> KIDS_OTT[Перехід на OTT-платформу]
    KIDS_OTT --> SUCCESS([✅ Успіх: дитина дивиться безпечний контент])

    KIDS_HOME --> EXIT_TAP[Тап «Вийти» у навігації]
    EXIT_TAP --> EXIT_PIN[PIN-екран: підтвердження виходу]
    EXIT_PIN --> EXIT_CHECK{PIN\nправильний?}
    EXIT_CHECK -->|так| ADULT[Головна — дорослий режим]
    EXIT_CHECK -->|ні| EXIT_ERR[Стан: неправильний PIN]
    EXIT_ERR --> EXIT_PIN
    ADULT --> EXIT_OK([✅ Успіх: вийшов — дорослий режим активний])
```

## J3 — Звуження без перегляду всього підряд

> Персони: Оксана (5 хв на вибір), Ірина («не гортати 20 хвилин»).
> Два шляхи всередині Пошуку: текстовий запит і панель фільтрів. Обидва ведуть до списку результатів.

```mermaid
flowchart TD
    START([Знаю приблизно що хочу — жанр або настрій]) --> SEARCH[Пошук]

    SEARCH --> MODE{Що вводить?}
    MODE -->|щойно відкрила — нічого| HINT[Стан: порожній пошук — підказки і популярне]
    HINT --> MODE
    MODE -->|назву фільму| TEXT[Вводжу текстовий запит]
    MODE -->|жанр або рейтинг| FILTERS[Відкриваю панель фільтрів]

    TEXT --> TEXT_R{Є результати?}
    TEXT_R -->|так| RESULTS[Список результатів у Пошуку]
    TEXT_R -->|ні — можлива помилка| SUGGEST[Стан: empty — пропозиція схожих назв]
    SUGGEST --> FIX[Виправляю запит]
    FIX --> TEXT
    SUGGEST --> BAIL_TEXT([⚠️ Тупик: назва не знайдена — пішла на Головну])

    FILTERS --> APPLY[Вибираю жанр / рік / рейтинг / вікову категорію]
    APPLY --> FILTER_R{Є результати?}
    FILTER_R -->|так| RESULTS
    FILTER_R -->|ні| EMPTY[Стан: empty — нічого під ці фільтри]
    EMPTY --> RELAX{Що робить?}
    RELAX -->|знімає один фільтр| APPLY
    RELAX -->|скидає всі| SEARCH
    RELAX -->|кидає| BAIL_FILTER([⚠️ Тупик: не знайшла — пішла гортати Головну])

    RESULTS --> SCAN{Щось\nпідходить?}
    SCAN -->|так| TAP[Тап на постер]
    SCAN -->|ні — весь список не той| CHANGE[Змінюю або додаю фільтр]
    CHANGE --> APPLY
    TAP --> CARD[Картка фільму]
    CARD --> SUCCESS([✅ Успіх: знайшла — рейтинг переконав])
```
